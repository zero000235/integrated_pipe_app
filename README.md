# 🚀 초통합 프리미엄 배관 시스템 시뮬레이터 (Pipe System Integrated Simulator)

> **수학적 완벽성과 인터랙티브 CAD 드로잉이 결합된 차세대 유체 시스템 설계 플랫폼**

본 프로젝트는 배관 공학에서 가장 널리 쓰이는 **하디-크로스(Hardy-Cross) 수리 유동 해석 솔버**와 **정밀 압력강하(수두 손실) 수력학 설계 모형**을 완벽하게 통합한 웹 기반의 초통합 설계 플랫폼입니다. 사용자가 격자판(Grid) 위에서 배관 요소를 직관적으로 그리고 사양을 입력하면, 물리 엔진이 즉각적으로 계통 내 유량, 펌프 설계 전양정, 경제적 KS 관경, 그리고 20년 생애주기 총비용(LCC) 견적서를 실시간으로 도출해 냅니다.

---

## 🌟 핵심 고도화 기능 (Core Highlights)

### 1. Spanning Tree 기반 독립 폐회로 추출 및 Signed Flow 수리 해석
* **신장 트리(Spanning Tree) 모델**: 복잡한 다중 폐회로 관로망에서 기하학적으로 선형 독립인 최소 독립 루프(Fundamental Loops)만을 그래프 이론에 근거하여 100% 자동으로 정확히 검출합니다.
* **Signed Flow 솔버**: 유량의 크기뿐 아니라 물리적 흐름 방향(부호, $+$ 및 역방향 $-$ 부호)을 음수 영역까지 자연스럽게 수렴하도록 풀어내어, 임의의 복잡한 그물망식 루프 관로에서도 완벽한 질량 보존(연속 방정식) 및 에너지 평형을 만족시킵니다.
* **수치 안정 가드**: 유동이 거의 없는 정지 구간($Q \approx 0$)에서의 수치적 발산(`NaN` 및 `ZeroDivisionError`)을 스무딩 보간식으로 방어하여 극한 상황에서도 솔버가 튕기지 않고 부드럽게 수렴합니다.

### 2. Genereaux 공식 기반 LCC 최적 관경(Economic Pipe Diameter) 설계
* **생애주기비용(LCC) 최적화**: 배관 자재 구매 가격(관경이 클수록 비쌈)과 펌프를 20년 동안 돌리는 데 필요한 전력 구동 비용(관경이 클수록 마찰 저항이 줄어들어 요금이 저렴해짐)의 절충점을 수학적으로 찾아내는 **Genereaux 수식**을 내장하였습니다.
* **상용 KS 규격 자동 매칭**: 도출된 최적 경제적 내경을 기반으로 **KS D 3576 스테인리스 강관** 및 **KS D 3562 압력 배관용 탄소강관** 규격 테이블을 검색하여 실제 상용화된 최적 파이프 외경과 스케줄(SCH) 두께를 자동 매치 제안합니다.

### 3. Critical Path 양정 역산 및 NPSH 공동현상(Cavitation) 정밀 진단
* **비재귀 BFS 최장 손실 경로 색출**: 재귀 DFS 호출로 인한 스택 오버플로우나 무한 재귀 루프 붕괴 위험을 완전히 차단하는 비재귀 BFS 경로 솔버를 구현하였습니다. 펌프 토출구부터 계통 최말단까지의 경로 중 수력 마찰 수두 손실이 가장 심한 **최악 경로(Critical Path)**의 손실압을 역산하여 펌프에 실제로 필요한 전양정(Total Head, $H$)을 공학적으로 설계합니다.
* **NPSH 처방전**: CoolProp 라이브러리를 연동해 유체 온도별 포화증기압을 고정밀 계산하고, 흡입 배관의 손실을 뺀 **유효 흡입 수두(NPSHa)**를 계산합니다. 펌프에 요구되는 필요 흡입 수두(NPSHr)와 대조하여 공동현상 위험도를 3단계 신호등 형태로 분류하고 구체적인 공학적 개선 조치 처방전(Recipe)을 제공합니다.

### 4. 오토캐드(AutoCAD) 스타일 격자판 및 동적 네온 유동 애니메이션
* **CAD 감성 2중 축선 격자판**: 미세 격자망(Minor)과 5배수 주요 격자망(Major)이 유기적으로 조합되어 실제 도면 위에서 설계하는 것과 같은 정밀한 스냅 마그네틱 드로잉 환경을 완성하였습니다.
* **네온 물방울 트레일(Trail) 애니메이션**: 흐르는 유량의 부호와 유속 크기에 실시간 정비례하여 네온 그린 물방울이 꼬리 잔상을 남기며 관로를 타고 부드럽게 흘러가는 수려한 유동 역학 비주얼을 제공합니다.

## 🛠️ 기술 스택 (Technology Stack)

* **Backend**: Python 3.8+
* **Frontend UI**: Streamlit, HTML5 Canvas, Vanilla CSS & Modern JS (ES6)
* **Fluid Properties**: CoolProp (표준 국제 유체 열역학 라이브러리)
* **Data Processing**: NumPy, Pandas, Pillow
* **AI Analysis (Option)**: Google Gemini API (Vision Model)

## 🚀 설치 및 실행 방법 (Installation & Run)

### 1. 패키지 설치
로컬 터미널을 열고 본 레포지토리를 클론한 뒤, 필수 패키지를 설치합니다.
```bash
pip install -r requirements.txt
```

### 2. 어플리케이션 가동
```bash
streamlit run integrated_pipe_app.py
```
실행 직후 웹 브라우저 창에서 **[http://localhost:8501](http://localhost:8501)** 주소로 자동 연동되어 시뮬레이터 대시보드가 열립니다.
